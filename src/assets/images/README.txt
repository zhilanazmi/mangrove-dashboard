PANDUAN MENEMPATKAN GAMBAR KLASIFIKASI MANGROVE
================================================

Letakkan gambar-gambar klasifikasi mangrove Anda di folder ini dengan nama file berikut:

- mangrove-1990.jpg (atau .png)
- mangrove-2000.jpg (atau .png)  
- mangrove-2010.jpg (atau .png)
- mangrove-2020.jpg (atau .png)
- mangrove-2025.jpg (atau .png)
- mangrove-2030.jpg (atau .png)

Format gambar yang didukung: .jpg, .jpeg, .png, .webp

Setelah menempatkan gambar, aplikasi akan otomatis menampilkan gambar yang sesuai ketika Anda memilih tahun di dashboard.
